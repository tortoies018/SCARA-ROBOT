# SCARA 机器人控制器 — 用户手册

## 环境准备

```bash
pip install -r PC_Host/requirements.txt
python PC_Host/main.py
```

## 功能操作

### 1. 连接机器人

| 步骤 | 操作 |
|------|------|
| 选择端口 | 下拉框选择对应 COM 口，点 ⟳ 刷新列表 |
| 选择波特率 | 默认 115200，与固件一致 |
| 连接 | 点 `连接` 按钮，状态显示 ● 已连接 |
| 断开 | 再次点击按钮断开连接 |

### 2. 机器人控制

| 按钮 | 功能 | 串口指令 |
|------|------|---------|
| 🏠 回零 | 执行回零序列（2 段式光电开关寻零） | `H` |
| ✏ 抬笔 | 舵机抬笔 | `P0` |
| ✏ 下笔 | 舵机下笔 | `P1` |
| ⏹ 停止 | 紧急停止所有运动 | `!` |
| ⟳ 查询 | 查询当前位置和状态 | `Q` |

### 3. 正运动学 (FK)

从关节角度计算末端位置。

```
输入 θ₁、θ₂ → 点「计算正解」→ 显示 x、y
点「运动 →」将机器人移动到计算出的位置（绝对坐标）
```

### 4. 逆运动学 (IK)

从笛卡尔坐标计算关节角度。

```
输入 X、Y (mm) → 点「计算逆解」→ 显示 θ₁、θ₂
点「运动 →」将机器人移动到输入的坐标（绝对坐标）
```

- 若目标位置超出机械臂工作范围，显示「不可达」

### 5. 轨迹规划

#### 绘制直线

1. 点选 `直线` 模式
2. 在画布上**依次点击**多个点，每两点之间自动生成直线插值
3. 蓝色线为实际轨迹，青色虚线为预览
4. 点 `预览` 生成 DDA 步进段

#### 绘制圆弧

1. 点选 `圆弧` 模式，勾选 `逆时针` 控制方向
2. **第一次点击**：设置圆心位置
3. **第二次点击**：设置弧线起点
4. **第三次点击**：设置弧线终点，自动生成圆弧插值

#### 参数设置

| 参数 | 范围 | 说明 |
|------|------|------|
| 速度 | 100 ~ 20000 | 步进频率 (Hz)，决定移动快慢 |
| 步长 | 0.1 ~ 10.0 mm | 插值精度，越小轨迹越平滑、段数越多 |

#### 执行轨迹

1. 点 `预览` 查看分段数
2. 点 `▶ 发送轨迹` 逐段下发到机器人
3. 进度栏显示 ✓ 完成 / ✗ 失败

### 6. 画布操作

| 操作 | 方式 |
|------|------|
| 缩放 | 鼠标滚轮 |
| 平移 | Ctrl + 左键拖动，或中键拖动 |
| 复位 | 点 `⊡ 复位` 按钮 |
| 十字光标 | 鼠标悬停显示精确坐标 |

### 7. 控制台

- 实时显示所有串口收发的原始指令
- 蓝色 `→` 表示发送，橙色 `←` 表示接收
- 底部输入框可直接发送原始串口指令

## 串口协议参考

| 指令 | 格式 | 说明 |
|------|------|------|
| 回零 | `H\r\n` | 执行 2 段式回零 |
| 抬笔 | `P0\r\n` | 舵机角度 0° |
| 下笔 | `P1\r\n` | 舵机角度 180° |
| 相对移动 | `M d1 d2 speed\r\n` | 电机1/2 步数 + 速度 |
| 绝对移动 | `A s1 s2 speed\r\n` | 目标绝对步数 + 速度 |
| 查询 | `Q\r\n` | 返回当前步数和状态 |
| 停止 | `!\r\n` | 紧急停止 |
| 设置位置 | `SP s1 s2\r\n` | 手动设置当前位置 |

## 机械参数

| 参数 | 值 |
|------|-----|
| 大臂长度 (L1) | 150 mm |
| 小臂长度 (L2) | 100 mm |
| 步进电机步数/圈 | 200 × 8 细分 = 1600 |
| 同步带减速比 | 5:1 |
| 关节步数/圈 | 8000 |
